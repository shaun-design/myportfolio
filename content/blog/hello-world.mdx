Welcome to my site! This is the first post on my new portfolio and blog, and I figured a good place to start is explaining what this space is for and who it's aimed at.

## What to Expect

I plan to write about:

- **Design** — user experience, visual design, and design systems
- **Development** — React, Next.js, TypeScript, and building for the web
- **Process** — how I approach problems, ship work, and think about craft

Posts will range from short observations to longer deep-dives. I'm not aiming for a rigid schedule — just writing when I have something worth sharing.

## About This Site

This site is built with [Next.js](https://nextjs.org) and the App Router, styled with [Tailwind CSS](https://tailwindcss.com), and uses [shadcn/ui](https://ui.shadcn.com) components. Blog posts and case studies are written in MDX — Markdown with optional React component support — stored as plain files in the repo.

That means no database, no CMS, no build pipeline beyond `git push`. Simple on purpose.
